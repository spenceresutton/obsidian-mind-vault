# SJN Inpatient UTI Guidelines

## Introduction

These guidelines prioritize reducing IV catheter-related adverse events, costs, and resource burden. Core operational principles:

- IV therapy reserved for: patients unable to take PO, susceptibility profiles precluding oral agents, critically ill patients, or tenuous hemodynamics
- Transition to oral therapy when feasible
- Antimicrobial choice adjusted based on urine culture and susceptibility results
- Bacteria on UA alone does not confirm infection — urogenital flora is common
- Acute mental status change in older patients WITHOUT genitourinary symptoms or systemic signs is **NOT** indicative of a true infection

---

## Definitions

| Type | Definition |
|------|-----------|
| Asymptomatic Bacteriuria (ASB) | Positive urine culture WITHOUT any signs or symptoms of a UTI |
| Uncomplicated UTI (Cystitis) | UTI localized to the bladder AND positive urine culture |
| Febrile or Bacteremic UTI | UTI extending beyond the bladder with systemic symptoms AND positive urine culture |
| Complicated UTI | Febrile UTI, UTI with severe sepsis, UTI with bacteremia, pyelonephritis, or UTI with obstruction |
| CAUTI | Presence of an indwelling urinary catheter AND signs or symptoms of UTI |
| Pyelonephritis | UTI that has ascended into the kidney |

**New classification context:** SJN aligns with updated terminology — "uncomplicated" = bladder-confined in afebrile women or men; "complicated" = extending beyond the bladder. Prostatitis is NOT covered by these guidelines.

---

## Asymptomatic Bacteriuria (ASB)

**Do NOT treat in most circumstances**, regardless of pyuria, bacterial density, or resistant organism isolation.

**Treat ONLY in:**
- Pregnancy
- Prior to urologic procedures

**Pregnancy prescribing restrictions:**
- Fluoroquinolones and doxycycline: relatively contraindicated throughout pregnancy
- TMP/SMX: avoid in first 8 weeks of pregnancy

| Agent | Dose | Duration |
|-------|------|----------|
| Nitrofurantoin | 100mg PO BID (CI if CrCl <30 mL/min) | 5 days |
| Cephalexin | 500mg PO q6h | 7 days |
| Fosfomycin | 3g PO once^≠ | One dose |

## Uncomplicated UTI (Cystitis)

Use empiric options unless prior microbiological data reveals recent resistant organism history.

### Empiric Therapy (covers Enterobacterales)

| Agent | Dose | Duration |
|-------|------|----------|
| Nitrofurantoin | 100mg PO q12h (CI if CrCl <30 mL/min) | 5 days |
| TMP/SMX | 1 DS PO q12h | 3 days |
| Single-dose aminoglycoside (choose one): | | One dose |
| — Gentamicin | 5 mg/kg IV/IM | |
| — Tobramycin | 5 mg/kg IV/IM | |
| — Amikacin | 15 mg/kg IV/IM** | |
| Amoxicillin/clavulanate | 875-125mg PO q12h | 3-5 days |
| Cephalexin | 500mg PO q12h | 3-5 days |
| Cefpodoxime | 200mg PO q12h | 3-5 days |
| Levofloxacin | 250-500mg PO q24h (750mg if MDRO) | 3 days |
| Ciprofloxacin | 500mg PO q12h | 3 days |
| Ceftriaxone | 1g IV q24h | 3 days |

**Fluoroquinolone note:** Not first-line due to high E. coli resistance rates and collateral damage risk (resistance, C. difficile). Reserve when other PO options are not feasible.

### Culture-Defined Therapy

| Organism | Agent | Dose | Duration |
|----------|-------|------|----------|
| *P. aeruginosa* | Levofloxacin | 750mg PO q24h | 3 days |
| | Ciprofloxacin | 500mg PO q12h | 3 days |
| | Tobramycin | 5 mg/kg IV/IM (single dose) | One dose |
| | Amikacin** | 15 mg/kg IV/IM (single dose) | One dose |
| *Enterococcus* spp. | Nitrofurantoin | 100mg PO q12h (CI if CrCl <30) | 5 days |
| | Amoxicillin | 500mg PO q8h | 3-5 days |
| | Linezolid** | 600mg q12h | 5 days |
| *E. coli* & *E. faecalis* ONLY | Fosfomycin^≠ | 3g PO | One dose |
| Yeast | No treatment in most patients | | |

**Fosfomycin note:** Use only for *E. coli* or *E. faecalis*. Less effective than nitrofurantoin — use only if nitrofurantoin is contraindicated.

**Yeast note:** Typically urogenital flora, rarely a true pathogen. Consult ID if treatment need is uncertain.

## Complicated UTI

Includes: febrile UTI, severe sepsis, bacteremia, pyelonephritis, obstruction.

**Important:** Asymptomatic bacteriuria with or without pyuria in catheterized patients is NOT an indication for treatment.

**Nitrofurantoin and fosfomycin should NOT be used** in complicated UTI given upper-tract or parenchymal involvement concerns.

### Empiric Therapy (covers Enterobacterales)

| Scenario | Agent | Dose | Duration |
|----------|-------|------|----------|
| Without known prior resistance or septic shock | Ceftriaxone | 2g IV q24h | 7 days |
| Critically ill or septic shock | Cefepime | 2g IVPB q8h | 7 days |
| | OR Piperacillin/tazobactam | 4.5g IVPB q8h extended infusion | 7 days |
| Known/recent ESBL history | Meropenem** | 1g IVPB q8h | 7 days |
| | OR Ertapenem | 1g IV q24h | 7 days |
| Oral option | TMP/SMX | 1 DS PO q12h | 7 days |

**AmpC organisms** (*Enterobacter cloacae*, *Klebsiella aerogenes*, *Citrobacter freundii*): treat empirically with cefepime; switch PO to non-beta-lactam if susceptible.

### Culture-Defined Therapy

| Organism | Agent | Dose | Duration |
|----------|-------|------|----------|
| Enterobacterales | Levofloxacin | 500mg PO q24h | 5-7 days |
| | Ciprofloxacin | 500mg PO q12h (750mg if ESBL or AmpC) | 5-7 days |
| | Amoxicillin/clavulanate | 875-125mg PO q12h | 5-7 days |
| | Amoxicillin/clavulanate (bacteremic) | 875-125mg PO q8-12h | 7 days |
| | Cephalexin | 500mg PO q6h | 5-7 days |
| | Cephalexin (bacteremic) | 1000mg PO q6h | 7 days |
| | Cefpodoxime | 400mg PO q12h | 5-7 days |
| | Levofloxacin | 750mg PO q24h | 5-7 days |
| *P. aeruginosa* | Pip/tazo | 4.5g IVPB q8h extended infusion^° | 7 days |
| | Cefepime | 2g IVPB q8h | 7 days |
| | Meropenem** | 1g IVPB q8h | 7 days |
| *Enterococcus* spp. | Amoxicillin | 1000mg PO q8h | 7 days |
| | Linezolid** | 600mg PO q12h | 7 days |
| Yeast | No treatment in most patients | | |

## Catheter-Associated UTI (CAUTI)

**Scope:** Without symptoms indicating infection beyond the bladder (fever, severe sepsis, pyelonephritis, bacteremia, obstruction).

**CAUTI definition:**
- Indwelling urinary catheter in place >2 consecutive days in an inpatient location on the date of event (present any portion of the day OR removed the day before), AND
- At least one of: fever >38.0°C, suprapubic tenderness, costovertebral angle pain/tenderness, urinary urgency, frequency, or dysuria

**Key rule:** If catheter removed and symptoms resolved, likely no infection was present — no further treatment needed.

| Agent | Dose | Duration |
|-------|------|----------|
| Nitrofurantoin | 100mg PO q12h (CI if CrCl <30 mL/min) | 7 days |
| TMP/SMX | 1 DS PO q12h | 7 days |
| Cephalexin | 500-1000mg PO q8h | 7 days |
| Levofloxacin | 500mg PO q24h | 7 days |
| Ciprofloxacin | 500mg PO q12h | 7 days |

## Formulary and Restriction Notes

| Symbol | Meaning |
|--------|---------|
| ** | Restricted antimicrobial — requires Infectious Disease approval |
| ^≠ | Non-formulary — requires non-formulary medication request form |

**Restricted antimicrobials:** Amikacin, Linezolid, Meropenem

**Non-formulary:** Fosfomycin

## Key SJN-Specific Teaching Points for Case Design

1. **ASB is almost always a trap** — the correct answer is no treatment. The only exceptions are pregnancy and pre-urologic procedure. Acute mental status change alone in an elderly patient does NOT qualify.

2. **Nitrofurantoin and fosfomycin are upper-tract contraindicated** — both are bladder agents only. Complicated UTI, pyelonephritis, or bacteremia → do not use. This is an explicit SJN rule, not just general practice.

3. **Single-dose aminoglycoside is a first-line empiric option for uncomplicated cystitis** — gentamicin 5mg/kg or tobramycin 5mg/kg as a single IV/IM dose. Amikacin requires ID approval. This is a specific SJN option tested in UTI-07.

4. **AmpC organisms default to cefepime** — *Enterobacter cloacae*, *Klebsiella aerogenes*, *Citrobacter freundii* should be treated empirically with cefepime. PO switch to non-beta-lactam if susceptible. Don't use third-gen cephalosporins.

5. **Septic shock UTI escalation** — cefepime 2g q8h or pip/tazo 4.5g q8h extended infusion. ESBL history → meropenem or ertapenem.

6. **Fluoroquinolones are not first-line** — explicitly reserved when other PO options are not feasible, due to E. coli resistance and C. difficile risk.

7. **CAUTI: catheter removal is the intervention** — if catheter removed and symptoms resolve, no antibiotics needed. Asymptomatic bacteriuria in catheterized patients is NOT an indication for treatment.

8. **Fosfomycin organism scope is narrow** — *E. coli* and *E. faecalis* only. Not for other Enterococcus, not for Pseudo. Non-formulary at SJN.

9. **Complicated UTI duration is 7 days** — consistent across all empiric and culture-defined regimens. Uncomplicated cystitis ranges from 1 dose (aminoglycoside) to 5 days (nitrofurantoin). Duration selection is a tested variable.

10. **Vancomycin is not a UTI agent** — Enterococcus is covered with amoxicillin or linezolid (restricted). VRE coverage references ampicillin/linezolid, not vancomycin.
